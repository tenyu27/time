# PST Clock Overlay

A premium, modern, and lightweight clock overlay for OBS. Designed for clarity and aesthetic appeal with a glassmorphism effect.

Hosted at: [pst.tenyu.gg](https://pst.tenyu.gg)

## Setup in OBS
1. Create a new **Browser Source** in OBS.
2. URL: `https://pst.tenyu.gg`
3. Width: `300`
4. Height: `100`

## URL Parameters
Customize the clock by appending these to the URL:

- `?12h=true`: 12-hour format.
- `?scale=2`: Change size without losing quality.
- `?transparent=true`: Removes the glass background.
- `?style=...`: Apply custom CSS.
